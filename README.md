# MicroIntern Portal

AI-powered Micro-Internship Portal for students, organizations, and administrators.

## Overview

MicroIntern Portal is a modern SaaS-style internship management platform that helps students discover micro-internships, receive AI-powered recommendations, track applications, build professional profiles, and connect with organizations through a centralized system.

## Installation

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Run lint:

```bash
npm run lint
```

## Environment Variables

VITE_FIREBASE_API_KEY=
